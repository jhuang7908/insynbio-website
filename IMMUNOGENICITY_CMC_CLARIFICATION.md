# 免疫原性与CMC评估的定位说明

## 核心定位

### 免疫原性评估（Immunogenicity Assessment）

**当前状态**：
- ❌ **没有运行IEDB / NetMHC / PBMC**
- ✅ **基于**：
  - 人源化比例
  - Germline使用情况
  - 非人残基数量
  - Hallmark/Vernier控制情况

**本质**：**风险分级（Risk Stratification）**，不是概率预测（Probability Prediction）

### CMC / Developability评估

**当前包含**：
- 非典型Cys → warning
- 长CDR3 → risk
- FR2疏水暴露控制

**不包含**：
- 实际Tm
- 聚集动力学
- 表达量预测

**本质**：**工程可制造性启发式评估（Engineering Developability Heuristic Assessment）**

## 标准表述（用于报告）

### 免疫原性评估

**正确表述**：
```
本报告中的免疫原性评估基于人源化程度、种系使用情况和结构风险指标，属于工程级风险评估（Engineering Risk Assessment），而非基于T细胞表位预测的计算免疫原性评估。

当前评估包括：
- 人类种系使用情况（是否使用高频人类germline）
- 非人残基数量与位置（FR区是否残留camelid特异位点）
- Hallmark/Vernier区域控制情况（是否避免暴露疏水界面）
- 异常结构信号（超长CDR3、非成对Cys、非典型FR2）

这些指标在工业界通常称为"Developability & Immunogenicity Heuristic Assessment"。

基于T细胞表位预测的免疫原性评估（如IEDB MHC-II预测）可作为可选模块，在用户明确要求或指定HLA panel时进行。
```

### CMC/可开发性评估

**正确表述**：
```
本报告中的CMC/可开发性评估基于序列的工程风险指标，属于启发式评估（Heuristic Assessment），而非实际物理化学性质预测。

当前评估包括：
- 序列风险基序识别（N-糖基化、脱酰胺化、异构化、氧化位点）
- 聚集风险斑块指标（疏水斑块、电荷斑块）
- 综合可开发性评分（基于风险因素扣分）

这些指标用于快速识别潜在的聚集、降解和稳定性问题，但不能替代实际的：
- 热稳定性实验（Tm值）
- 聚集动力学实验
- 表达量实验

实际的可制造性评估需要通过湿实验验证。
```

## 为什么现在不跑IEDB？（专业解释）

### 四个原因

1. **IEDB对VHH的假阳性非常高**
   - VHH很短（~110 aa）
   - FR/CDR高度工程化
   - 很多预测到的"强binder"在真实PBMC里是阴性
   - 工业界不会在早期人源化就过度依赖IEDB

2. **IEDB需要明确的HLA背景**
   - IEDB必须指定HLA-DR/DP/DQ等位基因
   - CRO客户通常还没定人群
   - 在没有HLA背景时跑IEDB，本身就不严谨

3. **IEDB ≠ 可交付CRO证据**
   - IEDB结果不是regulatory-grade
   - 不能替代PBMC
   - 放在报告里反而容易被问"你们有没有做PBMC？"
   - 现在不放，比放了更安全

4. **软件层面的理性分层**
   - P0：切分/人源化/规则突变 ✅ 已完成
   - P1：工程风险评估 ✅ 已完成
   - P2：IEDB/epitope ❌ 刻意未启用
   - P3：PBMC ❌ 实验

这是正确的产品路线。
