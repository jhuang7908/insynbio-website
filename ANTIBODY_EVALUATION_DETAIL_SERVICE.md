# 抗体评估明细服务

**版本**: v1.0 | **日期**: 2026-02-24  
**交付方式**: **暂时人工服务**（由 InSynBio 专家执行并交付）

---

## 一、服务概述

抗体评估明细是针对单个或多个抗体的**综合评估报告**，整合序列、结构、可开发性、界面与免疫原性等多维指标，输出结构化的明细表格与专家解读。

| 项目 | 说明 |
|------|------|
| **输入** | VH/VL 序列、PDB（可选）、CDR 界定（可选）、抗原链 ID（Ab-Ag 复合物时） |
| **输出** | Markdown 或 PDF 评估明细报告 + JSON 结构化数据（可选） |
| **交付周期** | 依项目规模，通常 3–5 个工作日 |

---

## 二、评估明细内容

### 2.1 序列维度

| 模块 | 明细项 |
|------|--------|
| **编号与切分** | IMGT/Kabat 编号、FR/CDR 残基列表 |
| **Germline** | 最近胚系、identity%、VH/VL 黄金配对评估 |
| **可开发性** | pI、GRAVY、不稳定指数、净电荷、SAP 聚集倾向 |
| **TAP** | Total CDR Length、PSH、PPC、PNC、SFvCSP（含临床阈值对照） |
| **CDR 化学风险** | 脱酰胺、异构化、氧化、糖基化、游离 Cys 位点列表 |
| **免疫原性** | MHC-II 27 等位基因风险位点、聚类、表面免疫原性 |

### 2.2 结构维度

| 模块 | 明细项 |
|------|--------|
| **13 参数** | VH-VL 夹角、界面接触对数、Vernier SASA、CDR 构型、pLDDT 等 |
| **界面解析** | BSA、Paratope/Epitope 残基列表、H-键、盐桥、疏水接触、SC 分数 |
| **阻断分析** | 与 PD-1/PD-L1 等靶点结合位点重叠度（若适用） |

### 2.3 多抗体比较（可选）

| 模块 | 明细项 |
|------|--------|
| **表位重叠** | 共同/独有 epitope、竞争关系 |
| **界面对比** | BSA、SC、接触数、阻断潜力排序 |

---

## 三、报告示例结构

```markdown
# <抗体名称> 评估明细

## 1. 序列信息
- VH/VL 序列、编号、FR/CDR 界定

## 2. Germline 与框架评估
- 胚系身份、黄金配对、临床先例

## 3. 可开发性明细
- pI、GRAVY、TAP、CDR 化学风险表格

## 4. 免疫原性明细
- 风险位点、聚类、建议

## 5. 结构评估明细（若有 PDB）
- 13 参数、界面、Paratope/Epitope、阻断分析

## 6. 多抗体比较（若有）
- 表位重叠、BSA/SC 对比

## 7. 局限与不确定性说明
- 预测结构、cutoff 等影响
```

## 四、挂网服务说明（网站用）

| 服务名称 | 交付方式 | 备注 |
|----------|----------|------|
| 抗体评估明细 | 暂时人工服务 | 3–5 工作日，需提供 VH/VL 序列，PDB 可选 |
