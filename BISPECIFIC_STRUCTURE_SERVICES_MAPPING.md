# 双特异抗体：结构分析 → 客户痛点 → 可提供服务

理清 ESMFold / ImmuneBuilder 分析结果能带来什么信息，以及如何对应客户需求。

---

## 一、结构分析工具与输出

### 1.1 ESMFold（scFv-like 单链）

| 输入 | 输出 | 可提取信息 |
|------|------|------------|
| scFv 序列（VH-linker-VL） | PDB + pLDDT | 结构置信度、CDR 构型、VH-VL 界面 |
| Tandem scFv（scFv1-linker-scFv2） | 整链 PDB | 两臂相对取向、linker 柔性、domain 间接触 |

**可计算指标**（需后处理，如 structure_metrics_humanization 或 BioPython）：

| 指标 | 含义 | 客户关心点 |
|------|------|------------|
| **pLDDT** | 预测置信度 | 结构是否可信、能否用于后续设计 |
| **VH-VL 夹角** | 界面几何 | 界面是否稳定、是否影响结合 |
| **Vernier packing** | 接触数 | 界面稳定性、是否易聚集 |
| **interface_min_dist** | 界面距离 | 接触是否紧密 |
| **CDR 构型** | canonical 类别 | CDR 是否合理、是否 outlier |

**scFv 特有**：单链、无 Fc，分析重点是 **VH-VL 界面** 和 **linker 两侧 domain 接触**。

---

### 1.2 ImmuneBuilder（IgG-like）

| 输入 | 输出 | 可提取信息 |
|------|------|------------|
| VH + VL（两链） | Fab PDB | 标准 Fab 结构 |
| 双抗：两 Fab 臂 | 需分别建模 | 两臂各自质量、可单独评估 |

**可计算指标**：

| 指标 | 含义 | 客户关心点 |
|------|------|------------|
| **pLDDT** | 预测置信度 | 结构是否可信 |
| **VH-VL 夹角** | 界面几何 | 界面稳定性 |
| **interface 指标** | 接触数、距离 | 界面质量 |
| **Vernier packing** | 接触数 | 稳定性、可开发性 |

**IgG-like 特有**：有 Fc，可评估 Fc 效应；双抗还需评估 **链错配**（KiH 正交性）、**两 Fab 臂相对取向**。

## 二、客户痛点与结构分析的关系

### 2.1 双抗设计阶段常见痛点

| 痛点 | 客户问题 | 结构分析能回答什么 |
|------|----------|---------------------|
| **结构稳定性** | 设计会不会表达差、易聚集？ | pLDDT、VH-VL 夹角、interface 距离、Vernier packing → 界面是否稳定 |
| **结合位点** | 两臂是否都能正确折叠、结合？ | CDR 构型、pLDDT 分布 → 各臂 CDR 是否合理 |
| **Linker 设计** | linker 长度/序列是否合适？ | Tandem scFv 建模 → 两 domain 是否碰撞、linker 是否过紧/过松 |
| **链错配** | KiH 双抗会不会错配？ | 需正交界面设计，结构分析用于验证设计后正确配对 |
| **可开发性** | 能否成药、CMC 风险？ | pI、疏水 patch、聚集倾向（需序列+结构） |
| **免疫原性** | 非人源序列风险？ | 免疫原性模块（需序列） |

### 2.2 结构分析能回答 vs 不能回答

| 能回答 | 不能直接回答 |
|--------|--------------|
| 结构是否合理、界面是否稳定 | 亲和力、特异性（需实验或 Ab-Ag 复合物） |
| CDR 构型是否 outlier | 是否结合靶点（需结合位点分析） |
| linker 是否导致 domain 碰撞 | 最佳 linker 序列（需实验筛选） |
| 可开发性相关结构指标 | 最终成药性（需实验） |

## 三、结构分析 → 可提供的服务

### 3.1 scFv-like 双抗（ESMFold）

| 服务 | 输入 | 输出 | 客户价值 |
|------|------|------|----------|
| **scFv 结构评估** | scFv 序列 | PDB + pLDDT + VH-VL 夹角 + interface 指标 | 设计前/后验证，结构是否可信 |
| **Tandem scFv 评估** | scFv1-linker-scFv2 序列 | 整链 PDB + 两臂相对取向 + 接触分析 | linker 设计是否合理、是否碰撞 |
| **可开发性** | 序列 + 结构 | pI、SAP、CDR 风险 | 早期 CMC 风险提示 |
| **免疫原性** | 序列 | 风险等级、位点、聚类 | 非人源序列风险参考 |

### 3.2 IgG-like 双抗（ImmuneBuilder）

| 服务 | 输入 | 输出 | 客户价值 |
|------|------|------|----------|
| **单臂 Fab 评估** | VH + VL（每臂） | 结构 + 13 参数 | 两臂各自质量 |
| **两臂对比** | 两 Fab 臂 | 结构差异、角度差异 | 双抗对称性、一致性 |
| **可开发性** | 序列 | pI、SAP、CDR 风险 | CMC 风险 |
| **免疫原性** | 序列 | 风险等级 | 免疫原性参考 |

### 3.3 若有 Ab-Ag 复合物（ColabFold 预测或晶体）

| 服务 | 输入 | 输出 | 客户价值 |
|------|------|------|----------|
| **界面解析** | 复合物 PDB | BSA、H-bond、盐桥、SC score | 结合质量、表位分析 |
| **阻断分析** | 复合物 + 参考表位 | 重叠位点 | 与已知抗体竞争位点 |

## 四、客户需求与服务的对应关系

| 客户需求 | 痛点 | 我们可提供 | 依赖工具 |
|----------|------|------------|----------|
| **TCE（CD3 重定向）** | scFv 或 IgG 臂稳定性、linker 设计 | scFv/IgG 结构评估、可开发性、免疫原性 | ESMFold / ImmuneBuilder |
| **双肿瘤靶点** | 两臂是否都能正确折叠 | 两 Fab 臂分别评估、结构对比 | ImmuneBuilder |
| **免疫共刺激** | IgG 格式、Fc 效应 | Fab 结构评估、可开发性 | ImmuneBuilder |
| **Linker 优化** | 长度、序列选择 | Tandem scFv 建模、domain 接触分析 | ESMFold |
| **设计验证** | 设计是否合理 | 结构 + 13 参数 + 可开发性评估 | 两者 |
